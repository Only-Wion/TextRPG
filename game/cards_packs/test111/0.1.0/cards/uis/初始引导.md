# 欢迎来到阳光公寓

你刚刚搬进阳光公寓104室，开始了新的合租生活。在开始探索之前，请先告诉我一些关于你的基本信息，这样你的室友们也能更好地了解你。

**请设定你的角色**：
1. **姓名**：你希望被称呼为什么名字？
2. **性别**：你是男性、女性，还是其他性别认同？
3. **年龄**：你今年多大？（建议20-35岁之间）
4. **工作**：你目前从事什么职业？
5. **爱好**：你平时喜欢做什么？

设定完成后，你的角色卡将会更新，游戏将正式开始。你会先见到你的室友赵明，他会带你熟悉公寓环境。

（系统提示：这些设定会影响后续对话选项和事件触发，请认真选择。）
