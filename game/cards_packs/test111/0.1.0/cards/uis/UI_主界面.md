## 主界面布局

**顶部状态栏**：
- 当前地点：显示玩家所在位置
- 时间：显示游戏内时间（第X天 早/晚）
- 精力：当前精力值/最大精力值
- 金钱：当前持有金额

**中央区域**：
- 地点描述：当前地点的详细描述和可交互元素
- 角色列表：当前地点可互动的角色头像

**底部导航**：
- 移动：切换到地点选择界面
- 任务：查看当前任务和进度
- 角色：查看所有角色信息和好感度
- 背包：查看物品和装备
- 设置：游戏设置和保存

**快捷操作**：
- 点击地点中的可交互物品直接触发互动
- 点击角色头像开始对话
- 拖拽物品到角色可以赠送礼物
