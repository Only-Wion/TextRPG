## 属性系统

* **魂力**：根本之源，决定等级与技能威力。
* **体力**：行动基础，每回合自动恢复（上限与恢复速度可随修炼提升）。
* **健康**：受伤、中毒会下降，为 0 时陷入危机。
* **心情**：影响事件判定成功率与修炼效率。与好友聚餐、欣赏美景可恢复。
* **气运**：隐藏属性，影响奇遇概率。可被重大事件或天材地宝改变。
* **财富**：**金魂币、银魂币、铜魂币**（1 金=10 银=1000 铜）。系统显示如：**财富：15 金 42 银 87 铜**。
